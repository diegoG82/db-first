Modellare la struttura di una tabella per memorizzare tutti i dati riguardanti delle auto usate messe in vendita da un concessionario.

############################
NOT NULL: IL CAMPO CON QUESTO VALORE NON PUO' AVERE VALORE NULL.
UNSIGNED: NON PUO' ESSERE UN VALORE NEGATIVO.
############################

DATABASE AUTO USATE ----->

id: INT- UNSIGNED - PRIMARY_KEY(UNIQUE - NOT NULL) - AUTO INCREMENT (SUFFICENTE NUMERO DI CIFRE - CHIAVE PRIMARIA - NESSUN VALORE RIPETUTO - NON PUO' ESSERE NULL E CON AUTO INCREMENT IL VALORE DELLA COLONNA AUMENTA AD OGNI AGGIUNTA DI UNA NUOVA RIGA)

brand: VARCHAR (20) - NOT NULL (QUALCHE CARATTERE IN PIU' PER MARCHE PARTICOLARI es: Hindustan Motors )
model: VARCHAR (30) - NOT NULL (QUALCHE CARATTERE IN PIU' PER MODELLI PARTICOLARI UNICI es: Aventador LP 750-4 Superveloce Roadster)
car_type: VARCHAR (15) (BERLINA, SPIDER, SW ECC..)
doors: TINYINT - UNSIGNED (3,4 OPPURE 5 PORTE E NON PUO' ESSERE NEGATIVO)
seats_number: TINYINT - UNSIGNED (SUFFICENTE PER COPRIRE ANCHE UN PULLMAN E NON PUO' ESSERE NEGATIVO )
horses: VARCHAR (4) UNSIGNED (4 CIFRE PER LE SUPERCAR TIPO BUGATTI VEYRON DA 1000+ CV)
fuel: VARCHAR (10) - NOT NULL (BENZINA GASOLIO METANO... 10 CARATTERI BASTANO)
euro: VARCHAR (10) (EURO 1, EURO 2 ECC..)
color: VARCHAR (15) - NOT NULL (QUALCHE CARATTERE IN PIU' PER COLORI PARTICOLARI UNICI es: Rosso Maranello)

production__year:YEAR (CONTIENE SOLO L'ANNO XXXX es: 2020)
previous_owners: TINYINT (CIFRA TRANQUILLAMENTE COPERTA CON QUESTO VALORE)
km: MEDIUMINT - UNSIGNED (CI SONO AUTOMEZZI CON 1M DI KM MA SONO RARI, DOVREMMO ESSERE ABBASTANZA COPERTI)

car_picture: TEXT (LINK AL PERCORSO DELL'IMMAGINE) - DEFAULT (PERCORSO SUL SERVER DELL' IMMAGINE DI DEFAULT)
description: TEXT (FINO A  65535 CARATTERI, SIAMO AMPIAMENTE COPERTI)
price: DECIMAL (10,2) (PER LE SUPERCAR CHE ARRIVANO A COSTARE ANCHE QUALCHE MILIONE)
discount: DECIMAL (10,2) (PER ESSERE EQUIPARATO AL PREZZO DI BASE MA VA BENE ANCHE (9,2) SEMPRE SE TRATTIAMO AUTO DI ALTISSIMO LIVELLO)
availability: TINYINT - DEFAULT(1) (CON IL VALORE 1 (TRUE) DI DEFAULT L'AUTO E' DISPONIBILE)
